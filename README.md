
# MEMORIES

A mern based app where a user can store some important memories related to him or others.




## Tech Stack

**Client:** React, 

**Server:** Node, Express

**DB:** MongoDB



  
## Features

- User can create a memory with texts and photographs
- user can Delete their memories
- User can update their memories
